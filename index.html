Connect to Notion
Orbit uses Notion as your sync database so profiles are available on every device, and Claude AI to generate personalised insights.
1
Create a Notion Integration — go to notion.so/my-integrations, click New Integration, name it "Orbit", and copy the Internal Integration Token.
2
Create a Notion Database — in Notion, create a new full-page database called "Orbit People". Then open it, click ··· → Connections → add your Orbit integration.
3
Copy the Database ID — it's the 32-character string in your database URL: notion.so/THIS-PART?v=...
Your token is stored only on this device. Orbit never sends it anywhere except directly to Notion's API.
